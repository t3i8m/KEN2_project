## Project 2-1 2024-2025 
### YINSH Game implementation
    We have the basic implementation of the board game Yinsh to add AI bots inn the next phase
### How to run:
Run the MainApp.java in the ui. 
### Packages:
#### engine:
    All the neccessary function with the rules of the game and movement and placement of the rings and coins
#### Game_Components.board:
    All the data types used by the engine are in this package
#### ui:
    The game GUI along with the runable MainApp.java file
